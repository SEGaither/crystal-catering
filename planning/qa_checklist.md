# QA Checklist

## Local package

- Zip extracts cleanly.
- Local repo can be initialized.
- Owner image exists.
- Planning files are preserved.

## Site build

- index.html, styles.css, script.js exist.
- Site loads locally.
- Mobile layout works.
- No console errors.
- Anchors work.
- Quote form labels and validation work.
- Phone link works.
- Email is correct.
- Chatbot opens/closes and only answers approved FAQ.
- Owner image displays in Meet Crystal section.
- No unsupported claims.

## SEO

- Title and meta description exist.
- One H1.
- Open Graph tags exist.
- Conservative JSON-LD exists.
- robots.txt exists.
- sitemap.xml exists.
- FAQ is visible semantic HTML.
- Image alt text exists.

## Pre-GitHub

- README still accurate.
- asset_manifest updated.
- implementation_notes updated.
- build_review updated.

## Post-Vercel

- Live URL loads.
- CTAs work.
- Mobile works.
- Metadata visible in source.
- No unsupported claims live.
